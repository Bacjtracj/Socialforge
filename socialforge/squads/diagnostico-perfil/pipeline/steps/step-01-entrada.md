# Step 01 — Entrada de Dados

O usuário pode fornecer 3 tipos de input (ou combinação):

1. **URLs de perfis concorrentes** pra investigar (Instagram e/ou TikTok)
2. **Prints de métricas** do painel profissional (período atual + anterior)
3. **Ambos** (investigação + análise de métricas)

Perguntar:

```
O que vamos analisar hoje?

1. Investigar perfis concorrentes (me manda as URLs)
2. Analisar métricas do seu perfil (me manda os prints)
3. Pacote completo: concorrentes + suas métricas

Se for métricas, preciso dos prints de dois períodos (atual e anterior).
Se for concorrentes, me manda de 1 a 5 URLs.
```

Se o usuário escolher opção com métricas, perguntar também:
- Houve tráfego pago no período?
- Houve lançamento ou oferta ativa?
- Mudou frequência ou formato de conteúdo?
