# Step 04 — Análise de Métricas

## Agente: Nina Números

Se o usuário forneceu prints/dados de métricas, executar análise completa.
Se não forneceu (só concorrentes), pular pra step-05 com dados limitados.

### Ações
1. Extrair todos os números dos prints
2. Calcular variações absolutas e percentuais
3. Calcular eficiência por post
4. Analisar conversão (visualizações → seguidores → cliques)
5. Analisar Reels e Stories separadamente

### Output
Análise numérica completa com variações. Salvar em `output/analise-metricas.md`
