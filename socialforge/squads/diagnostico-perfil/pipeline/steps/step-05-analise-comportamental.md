# Step 05 — Análise Comportamental

## Agente: Nina Números

### Ações
1. Identificar dias/horários dominantes
2. Identificar formato dominante (o que gera mais resultado real)
3. Identificar gatilho dominante (curiosidade, autoridade, prova social, polêmica, dor, transformação)
4. Diagnosticar posicionamento (crescimento sustentável / pico viral / conteúdo confuso / autoridade em construção)
5. Identificar o GARGALO PRINCIPAL (apenas um)
6. Entregar as 3 afirmações: repetir, parar, testar

### Output
Diagnóstico comportamental com gargalo. Salvar em `output/diagnostico-comportamental.md`
