# Step 02 — Pesquisa de Datas e Tendências

## Agente: Luna Pesquisa

### Ações
1. Usar web_search para buscar datas comemorativas do mês no Brasil
2. Buscar datas específicas do nicho do cliente
3. Identificar 3-5 tendências ou pautas quentes do setor
4. Classificar cada data por relevância (Alta / Média / Oportunidade) e tipo (Institucional / Educativo / Engajamento / Campanha)
5. Indicar quais datas têm potencial de campanha

### Output
Tabela de datas + lista de tendências. Salvar em `output/datas-comemorativas.md`
