# Step 12 — Entrega Final

Apresentar resumo de tudo que foi produzido:

1. Calendário editorial do mês (X posts)
2. Copy completo de cada post (ganchos, legendas, carrosséis, roteiros de Reels)
3. Roteiro de stories da semana (7 dias, 3 blocos por dia)
4. Score de qualidade do revisor

Perguntar formato de entrega preferido:
1. Markdown (no chat)
2. HTML interativo
3. Word (.docx)
4. Planilha (.xlsx)

Gerar o formato escolhido e entregar via present_files.
