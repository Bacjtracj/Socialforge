# Referência de Serviços — Adaptações por Tipo

## Gestão de Social Media

**Tom do manual:** Operacional e parceiro. O cliente contratou execução — ele precisa entender que o trabalho exige colaboração, não só pagamento.

**Seções obrigatórias adicionais:**
- Fluxo de aprovação de conteúdo (prazo de 48h, o que acontece se não aprovar)
- O que o cliente precisa entregar mensalmente (fotos, vídeos, informações de produtos/serviços, promoções)
- Política de alterações (quantas rodadas estão inclusas)

**Entregas típicas do plano:**
- Diagnóstico de perfil (mês 1)
- Planejamento estratégico mensal
- Criação de copy
- Criação de artes/criativos
- Agendamento e publicação
- Relatório mensal com análise

**Linguagem das entregas:** Direta, com prazos claros. Ex: "Todo mês, até o dia X, você recebe o planejamento do mês seguinte para aprovação. Você tem 48h úteis para responder."

**Regras que devem aparecer com destaque:**
- Prazo de aprovação de conteúdo: 48h úteis
- Material do cliente (fotos, vídeos, infos) deve chegar até o dia X do mês
- Alterações fora do escopo têm custo adicional
- Nenhum post sobe sem aprovação registrada

## Mentoria / Consultoria (Método Colmeia)

**Tom do manual:** Estratégico e transformador. O cliente comprou direcionamento — o manual deve posicioná-la como mentora com método, não como serviço disponível 24h.

**Seções obrigatórias adicionais:**
- "Sua trilha de evolução" — módulos ou encontros previstos, sequência lógica
- Regras de presença e aproveitamento (o que o cliente precisa fazer entre os encontros)
- O que NÃO está incluso (evitar expectativas equivocadas sobre suporte ilimitado)

**Entregas típicas:**
- Encontros individuais (frequência e duração)
- Acesso a materiais/gravações
- Canal de dúvidas entre encontros (com prazo de resposta)
- Diagnóstico inicial de posicionamento

**Linguagem das entregas:** Orientada a resultado e protagonismo do cliente. Ex: "Os resultados dependem diretamente da sua execução entre os encontros. Mentoria não é milagre — é método aplicado."

**Regras com destaque:**
- Cancelamento de sessão com X horas de antecedência
- Dúvidas fora dos encontros: respondidas em X dias úteis via [canal]
- Gravações disponíveis por X dias após o encontro
- O método só funciona se o cliente executar

## Produção de Conteúdo

**Tom do manual:** Criativo e processual. O cliente contratou produção — mas sem insumos dele, o trabalho para. O manual deve deixar isso explícito sem soar como cobrança.

**Seções obrigatórias adicionais:**
- Fluxo de produção completo: Diagnóstico → Estratégia → Copy → Criativos → Aprovação
- O que o cliente precisa fornecer (pautas, produtos, serviços do mês, fotos brutas)
- Prazo de entrega de insumos vs. prazo de entrega das peças prontas

**Entregas típicas:**
- Planejamento editorial mensal
- Criação de copy (quantidade de peças conforme plano)
- Criação de artes (formatos e quantidade conforme plano)
- Revisão inclusa (X rodadas)
- Entrega via [pasta Drive / plataforma de aprovação]

**Linguagem das entregas:** Clara quanto ao fluxo e dependências. Ex: "As peças são entregues em lotes. Para receber o lote da semana X, precisamos dos insumos até o dia Y."

**Regras com destaque:**
- Insumos entregues fora do prazo = entrega da semana postergada (sem cobrança adicional)
- Rodadas de alteração inclusas: [número]
- Alterações de conceito após aprovação de estratégia = fora do escopo
- Entrega e aprovação acontecem via [ferramenta/pasta definida]

## Serviço Combinado (mais de um tipo)

Quando o cliente contratar mais de um serviço:
1. Identifique o serviço principal (maior peso no contrato)
2. Use o tom desse serviço como base
3. Adicione as seções específicas dos demais serviços sem repetir o que já está no fluxo geral
4. Na seção "O que você contratou", liste todos os serviços separados por categoria com seus respectivos escopos
